```
msf> search [key:value, ...] regex
```



| Key  | Description                               | Example             |
| ---- | ----------------------------------------- | ------------------- |
| name | a descriptive name (aka a path segment)   | `search name:ftp`   |
| type | The type of Module (exploit, pivot, etc.) | `search type:pivot` |
| cve  | Search for a specific [[CVE]]             |                     |
