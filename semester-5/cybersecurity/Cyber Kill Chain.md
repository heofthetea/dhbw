1. **Recon** -> so viel Information über das Ziel finden wie möglich
	1. offizielle Quellen, mithören, [[Social Engineering]]
	2. Social Media kanäle/website anschauen (outside in scan)
2. **Weaponization** -> Lösungen entwickeln, wie man eindringen kann
3. **Delivery** -> Verteilen und Einschläusen von Exploit
4. **Exploitation** -> System tatsächlich knacken
5. **Installation** -> [[Backdoor]] installieren für dauerhaften Zugriff
6. **Command and Control** -> Kontrolle von außen über Ziel ergreifen
7. **Actions and Objective** -> das ursprüngliche Ziel tatsächlich erreichen
	1. Daten extrahieren & verkaufen
	2. Daten verschlüsseln -> [[Ransomware]]
