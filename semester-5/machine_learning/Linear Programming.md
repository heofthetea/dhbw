## Form
maximieren von:
$$f = \vec{c}^{t}\vec{x}$$
- wo:
	- $\vec{c}$ ein [[Vektor]] bekannter Koeffizienten
	- $\vec{x}$ der zu bestimmende Parameter
- unter nen Nebenbedingungen:

$$s.t.\quad A\vec{x} \leq \vec{b}$$
$$\land\quad \vec{x} \geq 0$$
- $A$: [[Matrix]] bekannter Koeffizienten
- $\vec{b}$: [[Vektor]] bekannter Koeffizienten

> [!hint] Das liefert ein [[LGS]]
