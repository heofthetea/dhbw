1. There exists a 2-dimensional [[Relation]] $\leq \subset M \times M$ for  [[Menge|Set]] $M$ ($M$ is "preordered")
2. $a \leq a$
3. $a \leq b\; \land \; b\leq a \; \Longrightarrow\; a = b$
4. [[transitiv|transitivity]] exists: $a \leq b \land b \leq c \Longrightarrow a \leq c$ 

> [!hint] Key difference from [[Total Order]]: $M$ may contain elements that are not comparable. e.g. intuitively: $\set{1, 2, 3, i}$ 
## Defined in Category
- Every Element is an [[Object]]
- A [[Morphism]] from $a$ to $b$ exists iff. $a \leq b$
	- thus: an [[Identity Morphism]] exists for every [[Object]]
- [[transitiv|transitivity]] is modelled by [[Composition]]
