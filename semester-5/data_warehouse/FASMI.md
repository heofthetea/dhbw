> Definition der Anforderungen für [[OLAP]]

- **Fast** -> Antwortzeit 5-20 Sekunden je nach Komplexität des Queries
- **Analysis** -> Klarer Fokus auf die Analyse, <span style="color:rgb(245, 154, 35)">intuitive</span> Nutzung möglich
- **Shared** -> Multi-[[User]] System, [[Role-based Access Control|Access Control]]
- **Multidimensional** -> [[Multi-Dimensional Data Model|MDDM]]; Multidimensionale View unabhängig von Data model
- **Information** -> Alle wichtige Information existiert


> [!hint] Der zentrale Punkt: [[Multi-Dimensional Data Model|Multi-Dimensionaliät]]
