### Garbage in, Garbage out
| information | ==> | Aussagequalität |
| ---- | ---- | ---- |
| 100% | Wissen | _sehr_ sicher |
| 90% | Schätzen | fast sicher |
| 50% | Schätzen | mittel sicher |
| 20% | Schätzen | sehr unsicher |
| 0% | Raten | keine Aussage |
