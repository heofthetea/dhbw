## Umbenennung einer Relation
$$R' := \rho_{R'}(R)$$
$$[R'] = [R]$$

- bennenne eine [[Relation]] um -> [[Schema einer Relation|Schema]] bleibt aber gleich.

```sql
SELECT Relation AS rel
```

oder

```sql
SELECT Relation rel
```
