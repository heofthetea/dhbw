## Was ist eine Mac Adresse?
[[MAC Adresse]]
- 6 Bytes
- Die ersten 3 Bytes beinhalten OUI (Hersteller)
	- dahinter laufende Nummer
- Die ersten 2 Bit:
	- individual/global
	- von IEEE
