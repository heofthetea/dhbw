> Bourne again shell

[[interpreted language]], [[Dynamic Typing]]

```bash
/bin/bash
```

- in wahrheit symlinked zu `/usr/bin/bash` und von dort aus weiter
- ist ein regular [[Executable]]
- wird [[Interpretation|interpretiert]] : Schlägt ein command fehl, wird das Skript trotzdem bis ans Ende durchgeführt

> [!hint] Oft ist `/bin/sh` ein [[Symlink]] auf `/bin/bash`.

> [!hint] Welche default shell ein user eingestellt hat, steht in `/etc/passwd`.
