[[Betriebssystem]]
- Schicht zwischen Anwendungssoftware und Firmware
## Aufgaben eines [[Betriebssystem|OS]]
1. **[[Computer]]-System nutzbar machen**
	1. verschiedene Hardware vereinigen
	2. Hardware schützen
	3. Netzwerk-Kommunikation
2. **[[Resource]]-Manager**
	1. [[Resource|Ressourcen]]-verteilung unter [[Prozess|Prozessen]] kontrollieren
	2. [[Virtual Memory Management]]
	3. Zugriffsrecheverwaltung

## Arten
1. [[Monolithisches Betriebssystem]] -> alles in einem Block
2. [[Geschichtetes Betriebssystem]] -> einzelne Schichten (vgl. [[Schichtenmodell]])
3. [[Mikroarchitektur]] -> Funktionalität wird komplett an [[Driver]] ausgelagert
4. [[Modulares Betriebssystem]] -> zentraler [[Kernel]] mit ladbaren Modulen
	1. Objektorientierter Ansatz -> [[Encapsulation]], Wiederverwendbarkeit & [[Interface|Interfaces]]
