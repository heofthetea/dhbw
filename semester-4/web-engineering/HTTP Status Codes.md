> A list of common status codes in [[REST]]



| Code | Name       | Info                                                                                     |
| ---- | ---------- | ---------------------------------------------------------------------------------------- |
| 200  | OK         | Commonn for Responses to [[HTTP GET\|GET]] Requests                                      |
| 201  | Created    | [[Object]] was created. Usually, the [[Object]] is returned.,                            |
| 204  | No Content | Responses to [[HTTP POST\|POST]] or [[HTTP PUT\|PUT]], where no content can be returned. |
| ...  |            |                                                                                          |
