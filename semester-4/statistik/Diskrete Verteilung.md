> Zufallsvariable, die nur **endlich** viele Werte annehmen kann

[[Wahrscheinlichkeit]]
$$f(i) = \begin{cases} P(X=x_{i}) = p_{i}\ \ f.\ x_{i} \in \set{x_{1}, x_{2},\dots,x_{k}}\\0\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \  \ \ \ \ \ else \end{cases}$$

Es gilt:
$$\sum\limits_{i=1}^{n}f(x_{i}) = 1$$
