> Seperate file containing _only_ [[CSS]].

### Link a sylesheet
```html
<link rel="stylesheet" href="my-format.css">
```

This is Metadata, thus belongs into an [[HTML head]].

## Content
```css
p {
	color: green;
	background-color: lightgray;
	text-align: center;
}
```


### multiple Values
- You can have _multiple_ values for one key!!!

```css
table, th, td {
	border: 1px solid;
}
```
